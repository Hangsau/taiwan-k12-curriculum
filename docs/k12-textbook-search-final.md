## 為什麼找不到？

**可能原因**：
1. **108 課綱後政策**：部編本幾乎廢除，教科書改由民間出版社編寫 + 教育部審定 — 出版權歸民間
2. **商業保護**：審定版教科書由康軒/翰林/南一/龍騰等出版，電子化僅供學校內部使用，不對外公開
3. **政府無電子化義務**：教育部沒要求教科書商提供公開電子版（只提供「教科書計價」登錄）
4. **學習扶助教材**有部分公開但屬於「弱勢輔導」性質，非完整課文

| 高中教材（學測/分科試題）| ✅ 163 個 PDF 已抓 |

| AI 自編國小教材 | ⚠️ 1 個示範（數學 1 年級 7000+ 字）|

**唯一可行路徑**：
1. **接受現實 + LLM 系統性自編**（每領域每年級 ~5000-10000 字 × ~60 組合 = 30-60 萬字 — 超 single session 容量）
2. **等 LLM API 帳號**做大批量生成（Anthropic API / OpenAI API）
3. **買商業授權**（不實際）
4. **用國際 OER 翻譯**（OpenStax 翻成中文 — 工作量更大）

**短期**：先用 AI 自編 1-2 個關鍵領域（國語文 / 數學），給下游用。
**長期**：等 LLM API 自動化整個 K12 教材生成。
