# 心測中心（國中教育會考）探索筆記（2026-06-24）

## 結論

**國中會考試題 PDF 沒公開下載**。只有「統計資料」「計分說明」相關 PDF 開放。

這與 NAER 教科書圖書館的結論類似：**政府機構有試題資料庫，但電子化純文字檔沒對外公開**。

## 探勘過程

### 1. 心測中心首頁

- 看到「歷屆試題」入口與「115/114/113/112/111 年國中教育會考」下拉選單

### 2. 歷屆試題頁
- **URL**：`https://cap.rcpet.edu.tw/examination.html`
- 下拉 select：option value 是 `exam/115/115exam.html` 格式
- 只顯示 5 年（111-115）

### 3. 各年度試題頁
- `/exam/115/115exam.html` 200 進得去

- **PDF 連結 0 個**（頁面只是入口目錄）

### 4. 各科目子頁
- `/exam/115/115exam1-1.html`（寫作）、`1-2.html`（國文）、`1-4.html`（數學）等

  - 「英語科計分方式說明.pdf」
  - 「數學科計分方式說明.pdf」
  - 「各科等級加標示與答對題數對照表.pdf」
  - 「各考區試務主辦學校.pdf」
  - 「報名作業說明.pdf」
  - 「隱私權聲明.pdf」「資安政策.pdf」

→ **沒有試題本身的 PDF**

### 5. 108-110 學年
- `/exam/108/108exam.html`、`/exam/110/110exam.html` 也 200 進得去
- 結構與 115 相同，PDF 同樣是統計/計分說明

## 為什麼找不到試題 PDF？

可能原因：
1. **心測中心政策**：試題本身（題本）只供紙本印製，PDF 僅供學校老師「在收到紙本後」核對，不對外公開
2. **著作權**：試題涉及命題老師，著作權歸心測中心，未授權電子散布
3. **商業考量**：試題印刷品有販售，公開 PDF 影響銷售
4. **未來政策**：可能逐步開放（國教署高中平臺有高中研究用試題）

## 給下個 session 的建議

- **不要浪費時間**再找國中會考「試題 PDF」— **政府沒公開**
- 改路徑：
  1. 找「基本學力檢測」（國小 5-6 年級，教育部辦，**可能有公開 PDF**）
