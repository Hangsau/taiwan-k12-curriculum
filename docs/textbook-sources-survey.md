## 2. 來源詳細評估

### 2.1 均一教育平台 ⭐ 首選

- **URL**：`https://www.junyiacademy.org/`
- **範圍**：K12 全年級、11 領域（涵蓋 108 課綱素養導向）
- **授權**：**CC BY-NC-SA 3.0 TW**
  - 商用禁止（NonCommercial）— 對我們的「教材遊戲」（非營利）**OK**
  - 修改後須相同授權（ShareAlike）— 我們 repo 用 CC BY-NC-SA 即可
  - 必須標示來源（Attribution）— `textbook/<...>.md` frontmatter 加 `source:` 欄位
- **內容形式**：教學影片 + 互動習題 + 講義（純文字 PDF / 內嵌文字）
- **已知內容**（首頁 + 課程列表抽樣）：
  - 「發現生活中的數學」— 數學素養導向
  - 「國中數學基礎概念」— 整數/分數/一元一次方程式
  - 「出發吧！探索這個世界」— 地理
  - 「理財素養」「半導體大冒險」「永續食代」— 跨領域素養
- **API 狀態**（2026-06-23 探勘）：
  - `/api/v2/coursemenu?depth=3` → 500（核心 menu API server-side 故障）
  - `/api/v2/header/btn_info` → 200（頂部按鈕設定，無關教材）
  - `/api/v2/alert_banner` → 200（公告，無關）
  - `/api/v2/user/profile` → 401（需登入）
  - `/api/v2/course` / `/topic` / `/category` 等 → 404
  - 結論：**完全沒公開教材 API**，必須 Playwright 模擬
- **Playwright 模擬路徑**（已知）：
  - 1. 進首頁
  - 2. 點「科目」按鈕（首頁 nav）→ 展開 menu
  - 3. 從 menu DOM 拿所有學科連結（不靠 API）
  - 4. 進學科頁 → 課程列表
  - 5. 進課程頁 → 抓影片網址 + 講義連結
  - 6. 對講義連結抓純文字（HTML 內 `<article>` 或 `<main>` 區塊）
- **預期困難**：
  - 影片是 YouTube 嵌入（網址可拿但內容要 ASR 或字幕檔）
  - 習題是互動式，純文字化需要展開 DOM 抓題目
  - 講義部分才有純文字（這是優先抓的）
- **預估工作量**：完整抓 K12 講義純文字 **20-40 hr**

### 2.2 國教署高中學科資源平臺 ⚠ 不是教科書

- **URL**：`https://ghresource.k12ea.gov.tw/nss/p/index`（自動 redirect）
- **範圍**：**普通型高中**（技術型 / 綜合型 / 單科型不在）
- **重要發現（2026-06-24）**：這是「**教師資源網**」不是「教科書下載站」
  - 教學資源庫 4192 筆都是「教案示例 / 教學設計 / 命題研發 / 試題研發 / 教學錦囊」
  - 9 個高中學科中心：國語文 / 英語文 / 數學 / 公民與社會 / 藝術生活 / 健康與護理 / 生活科技 / 資訊科技 / 全民國防教育
  - URL pattern：`/nss/p/{SubjectName}`（例：`/nss/p/Mathematics`）
  - 內容性質：**教師用的 how to teach，不是給學生看的課文**
- **仍有用**：教案中的「教學示例」「學習單」可作補充材料；評量試題可作習題
- **重新定位**：作為「**教案 / 試題補充材料**」來源（不是教科書主源）
- **不建議優先**：對「教材純文字」這目標效益低

## 4. 已知限制 / 取捨

- **影片內容**：均一教材含大量 YouTube 影片，純文字抓不到影片內容。v1 只抓**講義純文字**，影片留 URL 給後續 ASR 處理
- **互動習題**：均一習題是互動式（draggable / multiple choice），純文字化需要展開 DOM 抓「題目 + 選項 + 答案」，不是完整學習體驗
- **授權標示**：每份教材 .md frontmatter 必須有 `source:` 欄位（CC BY-NC-SA 要求 Attribution）
- **圖片 / 表格**：HTML 內的 `<img>` 和 `<table>` 純文字化會丟失 — 學習階段先抓文字骨架，圖表另存
- **PDF 教材**（高中平臺）：需要 pdftotext，可能版面順序亂（PDF 多欄排版），需要後處理
