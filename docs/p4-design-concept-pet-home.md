## 0. 核心迴圈（一句話）

**孩子學會一個課綱概念 → 寵物因此成長/進化、解鎖一件居家擺飾 → 孩子把擺飾佈置進自己的家、寵物住在裡面 → 想讓家更豐富/寵物更成長，就去學下一個概念。**

學習是因，寵物與擺飾是果。沒有任何一環靠「時間到了」「運氣」「不做就懲罰」來驅動。

---

## 1. 三大支柱 → SDT 三需求

這個玩法之所以值得做，是因為它能用三個天然載體，一次蓋住 SDT 三大需求（研究 §2.1）——這是純解鎖關卡（類型 C）做不到的：

| 支柱 | 對應需求 | 怎麼滿足 |
|------|---------|---------|
| **寵物養成** | 關聯 Relatedness | 孩子對生物產生情感依附（三需求中最難做的一個） |
| **居家擺飾佈置** | 自主 Autonomy | 自己決定家長什麼樣，是自我表達而非被指定（研究 §9.1 對高內在動機領域的建議方向） |
| **學習解鎖成長** | 勝任 Competence | 寵物進化/新擺飾都是「我學會了」的可見證明（mastery-coupled，研究 §5.3 / §6.1） |

---

## 2. 必須拆掉的兩個毒核

這兩種玩法各自帶一個「最上癮但最傷動機」的經典機制。**直接照搬 = 把框架最嚴重的反模式請進來**（framework §3.1）。設計第一要務就是拆掉它們。

### 2.1 拆掉電子雞的「疏忽懲罰」

- **地雷**：經典電子雞「不每天照顧就生病/死掉」——這是強制每日登入 + streak 中斷懲罰，framework Gate 3.2 命中兩條反模式，autonomy-thwarting（研究 §6.2 Duolingo streak 同理）。
- **替代設計**：
  - 寵物成長**只綁學習掌握度，不綁真實時間**（學會概念 → 進化；沒上線 → 牠原地等你，不退化、不生病、不死）。
  - 孩子離開再回來，寵物還在、狀態不變 → 守住「自然結束/合理暫停」（研究 §9.2-4）。
  - 想念寵物可以變成「回來學習」的拉力，但**絕不用罪惡感**當拉力。

### 2.2 拆掉居家擺飾的「稀有度炫耀」

- **地雷**：把擺飾做成 gacha 抽取 + 外顯稀有度（SSR 家具）+ 比誰房間有稀有貨——framework §3.1 兩條反模式，從自我表達掉進比較/炫耀。
- **替代設計**：
  - 每件擺飾都是**某個學習成就的紀念品**（靠學會東西解鎖，不靠抽、不靠運氣）。
  - 沒有稀有度標籤（不分 R/SR/SSR），差別只在「代表你學會了什麼」。
  - 社交以**互相參觀/分享佈置**為主，不做「房間排行榜」（合作非比較，研究 §9.2）。

## 3. 接課綱：成長/擺飾怎麼綁具體學習事件

核心原則（framework Gate 2 + 研究 §9.2-1/2）：**每個寵物進化、每件擺飾，都要可追溯到一個具體的課綱學習事件，並附「它代表你會了什麼」的說明**。資料來源是本 repo 的 `curriculum/` 結構化層。

舉例（概念示意，非最終對應表）：

| 領域 | 學習事件 | 解鎖物 + 內化說明 |
|------|---------|------------------|
| **數學** | 學會分數通分 | 擺飾「分數天秤」——卡片寫「你會把不同分母變成一樣的了」 |
| **自然科學** | 完成一個植物觀察單元 | 寵物解鎖「植物好友」型態 + 擺飾「小盆栽」——「你觀察到植物怎麼長大」 |
| **國語文** | 讀完並理解一篇故事 | 擺飾「故事書架」——「你讀懂了這個故事在說什麼」 |

擺飾/型態的清單之後在遊戲 repo 對著 `curriculum/<領域>/*.structured.json` 的學習內容編碼逐條對應（P3 知識圖譜完成後可自動化前備依賴排序）。

## 5. 上線前必須 A/B test 的證據缺口

本設計多數判定是 framework 的 `[推論]` 層，不是已驗證定論。上線前優先驗證：

1. **分離焦慮風險**（最高優先）：寵物擬人化依附是雙面刃。寵物「不會死但會想你」的設計，會不會反而讓低年級孩子焦慮？研究 §7 指出年齡層差異是 evidence gap，**無通過驗證的文獻可依賴**——這必須實測。
2. **年齡層分化**：研究 §7 / framework §3.3——低年級（1-3）對寵物依附可能最強，高年級（7-9）可能覺得幼稚甚至識破操弄。同一套寵物/擺飾要不要分齡呈現，要測。
3. **家長端**：若加家長可見儀表板，會不會把外在壓力灌進來（framework §3.2 不確定清單）——show「學了什麼」而非「贏了多少」，並 A/B test。

驗證手段（研究 §9.2-5）：SDT 三需求質性訪談 + long-term motivation durability 追蹤，不能只看 engagement 數字。
