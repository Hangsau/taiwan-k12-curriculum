# 考卷來源探勘報告

> **本檔在專案中的定位**
>
> §16-C 交付物：依 user 優先順序（教材 → **考卷** → 自編），這份報告盤點可用考卷來源。
> 用途：用考卷「分析內容 → 編排教材」。例如抓學測數學試題 → 看哪些 learning content code 反覆出現 → 那些 code 在教材優先加強。
>
> **狀態**：2026-06-24 初版

---

## 0. 一句話總結

**大考中心 (CEEC) 是首選** — 政府機構、完整試題 + 答案 PDF、可機讀（PDF → pdftotext）。
- 高中：學測 / 分科測驗 / 指考（110 前）/ 英聽 — 全部有
- 國小：基本學力檢測（教育部）+ 各縣市段考（公開版本有限）

---

## 1. 大考中心 CEEC ⭐ 首選

- **URL**：`https://www.ceec.edu.tw/`
- **授權**：政府機構 + 試題公開使用（學測/分科試題 PDF 公開下載）
- **試題範圍**（5 類考試）：
  - **學科能力測驗**（學測）：每年 1 月，高三生
  - **分科測驗**（110 前叫「指考」）：每年 6-7 月
  - **高中英語聽力測驗**：每年 1 月
  - **試辦考試**：新題型
  - **研究用試題**：研究用 PDF
- **可用內容**（2026-06-24 Playwright 探勘）：
  - **歷年試題及答題卷**：每年完整 PDF + 答案
  - **佳作**（作文/申論題範本）
  - **統計資料**：各科五標、級分對照
  - **參考試卷**：當年度示範
  - **研究用試題**：含詳解 PDF
- **下載連結範例**（實測）：
  - 一般試題：`https://www.ceec.edu.tw/xmfile?xsmsid=0J052424829869345634`
  - 特殊試題：`https://www.ceec.edu.tw/xmfile?xsmsid=0J052392083839398563`
  - 下載專區：`https://www.ceec.edu.tw/xmfile?xsmsid=0J052605494129871538`
- **抓法**：
  - PDF 直接 wget / curl 下載
  - pdftotext 轉純文字（pdftotext -layout）
  - 解析題目結構（題號 / 選項 / 答案）

---

## 2. 教育部國民小學基本學力檢測

- **URL**：搜尋「基本學力檢測」（每年 4 月 + 12 月實施）
- **範圍**：國小 5-6 年級國語文 + 數學 + 英語（部分年度）
- **授權**：政府公開
- **可用**：每年考古題 PDF
- **限制**：題數少（每科約 20-30 題），不如大考中心豐富

---

## 3. 政府開放資料平台 data.gov.tw ⭐ 國小考卷主源

- **URL**：`https://data.gov.tw/`
- **授權**：CC BY 4.0（政府開放）
- **搜尋關鍵字**：「教材」「課本」「學測」「段考」「基本學力」「試題」「評量」
- **可能 dataset**：
  - 各縣市段考考古題（部分縣市有公開）
  - 教育部「學習扶助」科技化評量系統資料
  - 大考中心歷年統計資料（已 JSON 格式）
- **優點**：API 可機讀、JSON 格式、不需 PDF 解析
- **缺點**：資料集分散、需要逐個搜尋

---

## 4. 縣市教育局段考（覆蓋率有限）

- 台北市教育局、新北市教育局等：部分公開段考題
- 多數縣市：段考由各校自命題，無公開版本
- 例外：教育部「國民中小學學習扶助」評量 — 公開

---

## 5. 教育雲考卷相關資源（教科書專區之外）

- 教育雲首頁有「**熱門資源**」「**教案**」「**學習單**」 — 部分教師自製 CC 授權
- 試題範例 / 教師研習影片 — 不算考卷，但有「教學示例」
- 進入 `https://market.cloud.edu.tw/`（教育大市集）→ 搜尋「段考」「評量」

---

## 6. 下載優先順序（給 §16-D 用）

按 user 「**教材 → 考卷 → 自編**」：

| 優先 | 來源 | 範圍 | 工作量 | 為何這優先 |
|------|------|------|--------|-----------|
| **1** | **大考中心 PDF** | 高中學測/分科/英聽/指考（110 前）| 2-4 hr | 完整公開、可機讀、政府授權 |
| **2** | **data.gov.tw** 搜尋 K12 試題 | 全 K12 試題 dataset | 2-4 hr | 政府開放、JSON 格式 |
| **3** | **國小基本學力檢測** | 國小 5-6 年級 | 1-2 hr | 政府公開、每年固定 |
| **4** | **教育雲「學習扶助」評量** | 國小 + 國中 | 1-2 hr | 政府公開 |
| 5 | LLM 生成考卷（自編）| 缺漏領域 | 取決於量 | 最後手段 |

**預估總計**：抓 5-10 年份考卷 → 分析哪些 curriculum codes 反覆出現 → 反推「關鍵學習重點」→ 編排教材優先順序

## 7. 已知限制 / TODO

- 大考中心只有高中試題，國小需要另尋（基本學力檢測 + 縣市公開段考）
- 段考多數由各校自命題，無公開版本
- 考卷 PDF 排版可能複雜（雙欄、表格、圖表），pdftotext 後可能需要後處理
- 解析題（作文 / 申論）無法純文字評分，只能抓「題目」用
